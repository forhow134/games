# 3D Games Collection

A collection of four 3D web games built with Three.js and Vite.

## Games

| Game | URL |
|------|-----|
| [Island Treasure](https://games.xgallery.online/island-treasure/) | Collect gems, avoid enemies |
| [Whack-a-Mole](https://games.xgallery.online/whack-a-mole/) | Classic whack-a-mole with 3D models |
| [Space Shooter](https://games.xgallery.online/space-shooter/) | Arcade space shooter |
| [Tower Defense](https://games.xgallery.online/tower-defense/) | Defend your temple against waves of enemies |

## Tech Stack

- **Three.js** - 3D rendering
- **Vite** - Build tool and dev server
- **JavaScript** - Game logic

## Development

```bash
# Install dependencies for a game
cd <game-directory>
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Deployment

Push to `main` branch. GitHub Actions automatically builds and deploys to GitHub Pages.
